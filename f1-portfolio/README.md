# 🏎️ Paddock.dev - High-Velocity Developer Portfolio

A sleek, responsive, and performance-driven software engineering portfolio template heavily inspired by Formula 1 telemetry and motorsport aesthetics. Designed for developers who want a fast, optimized, and aggressive visual identity.

## ✨ Features

- **The 10-Constructor Theme Switcher:** A custom-built CSS-variable engine allowing users to instantly switch the website's entire color palette, neon shadows, and accents between all 10 real-world F1 teams (Ferrari, Mercedes, Red Bull, McLaren, etc.).
- **High-Velocity UI/UX:** Sharp diagonal CSS clip-paths, checkered flag textures, and aggressive italicized typography mimicking high-speed physics.
- **Scroll Reveal Telemetry:** Elements and performance stats dynamically snap into view sequentially as the user scrolls down the straights.
- **Fully Responsive:** Fluid grid layouts ensure the portfolio renders impeccably on mobile devices, tablets, and ultrawide desktops.
- **Privacy First:** Pre-populated with dummy data ("Alex Vance" and fictional projects) so you can safely use it as an agency template without leaking personal project details.

## 🛠️ Technology Stack

- **HTML5:** Semantic architecture
- **CSS3:** Custom variables, grid layouts, fluid typography, and complex `clip-path` styling.
- **Vanilla JavaScript:** DOM manipulation for the Theme Switcher widget and scroll intersection observers.

## 🚀 How to Run

1. Clone this repository locally.
2. Navigate into the `f1-portfolio` directory.
3. Simply double-click `index.html` to open it in any modern web browser.
4. (Optional) Customize the dummy text inside `index.html` to reflect your own software engineering background.
